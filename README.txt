The source code in this assignment can be executed from the command line in the Terminal.

To run the program, locate the directory where the files are in, ensure the program is executable, and enter: ./Local_Alignment.py

	To ensure the program is executable, locate the directory where the files are in and enter: chmod +x Local_Alignment.py

	Otherwise enter: python Local_Alignment.py or python3 Local_Alignment.py

This program is a implementation of the Smith-Waterman algorithm to find best local alignment between two sequences. It will print
to the command line the alignment score and the best local alignemnt.
